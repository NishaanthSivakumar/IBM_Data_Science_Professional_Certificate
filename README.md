# IBM Applied Data Science Capstone Project

## Overview
This repository contains my work for the **IBM Applied Data Science Professional Certificate Capstone Project**.  
The project focuses on analyzing SpaceX launch data and building machine learning models to predict **Falcon 9 first stage landing success**.

---

## Project Workflow

1. **Data Collection**
   - SpaceX REST API  
   - Web scraping (Wikipedia)

2. **Data Wrangling**
   - Cleaning and transforming raw datasets  

3. **Exploratory Data Analysis (EDA)**
   - SQL queries and statistical summaries  
   - Identifying trends and patterns  

4. **Data Visualization**
   - Charts using Matplotlib, Seaborn, and Plotly  

5. **Geospatial Analysis**
   - Interactive maps using Folium  

6. **Predictive Analysis**
   - Logistic Regression  
   - SVM  
   - Decision Tree  
   - KNN  
   - Hyperparameter tuning using GridSearchCV  

---

## Key Insights

- Launch success has improved significantly over time  
- Launch site and orbit type influence landing outcomes  
- Payload mass shows correlation with landing success  
- Machine learning models achieved ~85–86% accuracy
